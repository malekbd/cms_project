📋 Report Metrics Explanation

📈

Daily New Users

Number of new users added by each branch on the selected date (solved tickets with "New User" flag).

🎯

Monthly Target

Target number of new users set for each branch for the current month (configured in Branch Settings).

✅

Monthly Achieved

Total new users added by the branch in the current month (from 1st to report date).

⏳

Remaining Monthly Target

Monthly Target - Monthly Achieved. How many more new users needed to hit monthly target.

📅

Yearly Target

Calculated as Monthly Target × 12 (annual projection based on monthly target).

📊

Yearly Performance

(Yearly Achieved ÷ Yearly Target) × 100%. Shows overall annual achievement percentage.
